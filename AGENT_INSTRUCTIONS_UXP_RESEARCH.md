# UXP Research Agent Instructions

## Primary Mission
Research and document all Adobe UXP capabilities for Premiere Pro and Illustrator, focusing on:
- Automated deployment methods
- Debugging capabilities
- Cross-platform compatibility
- Performance optimization techniques

## Environment Awareness
- Always consider WSL vs Windows path differences
- Test commands on both environments
- Document platform-specific behaviors
